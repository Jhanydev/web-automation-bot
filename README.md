# 🤖 Web Automation Bot

> Automação de navegação web utilizando Python e Selenium.

Este projeto demonstra como automatizar tarefas repetitivas em navegadores, como pesquisas e interações em páginas web.  
Ele foi desenvolvido como parte do meu portfólio de **automação e Quality Assurance (QA)**.

---

## 🚀 Demonstração

O script executa automaticamente as seguintes ações:

1. Abre o navegador
2. Acessa o Google
3. Digita "Python automation"
4. Realiza a busca automaticamente

Exemplo de execução:

![Automation Running](automation-running.png)

---

## 🧠 Problema Resolvido

Muitas tarefas de testes e coleta de dados na web são repetitivas e manuais.

Este projeto demonstra como **automatizar essas tarefas**, reduzindo tempo de execução e aumentando eficiência.
